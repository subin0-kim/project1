# 2. 핵심 로직 (Core Logic)

## 2.1 조작법 (Controls)
- **고정형 캐릭터**: 플레이어 캐릭터(무당/박수)는 이동하지 못하고 화면 정중앙에 고정되어 있습니다 ('인간 닻' 설정).
- **공격 조작**: 화면의 사방에서 밀려오는 요괴들을 향해 타이밍과 방향에 맞춰 화면을 **스와이프(Swipe)** 하여 공격(먹선 날리기)을 수행합니다.

## 2.2 스와이프 처리 (Swipe Handling)
- **입력 감지 시스템**: 화면 터치(Touch Down) 후 드래그(Drag) 시 시각적인 궤적(Trail)이 발생합니다.
- **벡터 계산**: 손을 떼는 순간(Touch Up), 시작점과 끝점 사이의 **방향 벡터(Vector2)** 를 계산합니다.
- **공격 판정**: 계산된 방향 벡터를 향해 플레이어의 공격 투사체(먹선)가 일직선 혹은 부채꼴 형태로 날아가 적을 타격합니다.

## 2.3 레벨업 로직 (Level Up)
- **혼불 획득**: 적을 처치하면 필드에 경험치 구슬인 '혼불'이 드랍되며, 이를 획득하여 경험치 게이지를 채웁니다.
- **로그라이크(Vampire Survivors-like) 스킬 선택**:
  - 레벨업 시 게임이 일시 정지되며 무작위로 3개의 아이템/기술(스킬 트리 참조) 선택지가 제시됩니다.
  - 플레이어는 이 중 하나를 선택하여 현재 게임 세션 동안 스킬을 조합하고 강화해 나갑니다.

## 2.4 강신(Gangshin) 로직
- **신력 게이지**: 플레이어가 적을 처치하고 정화할 때마다 별도의 '신력 게이지'가 차오릅니다.
- **발동 조건**: 신력 게이지가 100% 충전되었을 때, 화면을 **길게 누르기(Hold)** 혹은 **더블 탭(Double Tap)** 조작을 통해 강신 상태를 발동합니다.
- **발동 효과**:
  1. 기획된 짧은 애니메이션/이펙트(예: 색상 반전)와 함께 일정 시간 동안 플레이어는 **무적 상태**가 됩니다.
  2. 선택된 캐릭터(무당/박수)의 고유 필살기(전용 강신 스킬)가 즉시 시전되어 화면 내의 적들에게 궤멸적인 타격을 입힙니다.
